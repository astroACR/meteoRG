# meteoRG
Dashboard de estaciones meteorologicas basado en Google Cloud y Hosteado en Firebase.

https://meteorg-474500.web.app

La información meteorólogica utilizada proviene de tres instituciones:

  - DMC (Dirección Meteorólogica de Chile): API abierta al público, datos meteorologicos actualizados cada 15 mins.
  - INIA (Instituto Nacional de Investigaciones Agropecuarias): No existe API, se creó un endpoint manual, haciendo scraping del sitio.
  - NASA FIRMS (Fire Information for Resource Management System): API abierta al público, entrega puntos georreferenciados de posibles incendios, de manera global.
